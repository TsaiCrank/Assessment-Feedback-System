# EduConnect Assessment & Marking System (AMS)

A web-based assessment management platform developed for EduConnect learning centre using Agile Scrum methodology. The system digitizes and streamlines the entire assessment workflow from submission to grading and result release.

## Overview

This project replaces EduConnect's manual, paper-based assessment processes with a centralized digital solution that supports four user roles: Students, Instructors, Exam Administrators, and System Admins.

## Key Features

- **Student Portal**: View courses, submit assessments, receive notifications, and access grades with feedback
- **Instructor Portal**: Create and manage courses/assessments, grade submissions, and provide feedback
- **Exam Admin Portal**: Review and approve graded assessments, manage academic records, and view performance analytics
- **Admin Portal**: Manage users, assign roles, group students by intake, and control system access

## Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MySQL
- **Authentication**: JWT
- **Cloud Services**: SendGrid (notifications), Google Calendar API

## Project Details

- **Timeline**: 11 weeks (9 weeks development + 2 weeks deployment)
- **Methodology**: Agile Scrum (5 two-week sprints)

## Development Approach

Developed using Scrum framework with sprint-based incremental delivery, daily stand-ups, sprint reviews, and retrospectives. Deployed using accelerated phased rollout strategy over 2 weeks.
